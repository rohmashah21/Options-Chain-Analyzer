# Options Chain Analyzer
Analyze options chain data from Yahoo Finance to calculate Max Pain, Put Call Ratio, and identify key support/resistance levels.

## Features
- Fetch live options chain for any stock
- Calculate Put-Call Ratio (PCR) with sentiment (Bullish/Neutral/Bearish)
- Compute Max Pain the price where option sellers profit most
- Identify support (highest Put OI) and resistance (highest Call OI)
- Visualize IV Smile and Open Interest profile
